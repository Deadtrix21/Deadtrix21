### 🔭 What I'm Working On

#### 🔐 Cryptography & Low-Level Systems
*   **[ZNRG Engine (v1.0.0)](https://github.com/Deadtrix21/ZNRG-PUBLIC):** A high-performance encryption project built from the ground up, optimized for memory throughput and minimal binary footprint.
*   **Multi-Language Wrappers:** Maintaining custom bindings for the ZNRG library across Node.js (N-API), PHP (FFI), and C++.

#### 🏢 Organization Projects (@noxius-studium)
*   **eventflowsys:** Building robust event-driven system architectures.
*   **South African ID Validator:** Developing multi-language libraries for strict ID validation and data extraction.

#### 🎨 Modern UI / UX
*   **FluentFloatingIsland:** Developing a system-wide floating UI component for Windows. Leveraging WPF, MicaWPF, and Wpf.Ui controls to bring glassy, transparent, Wayland-style aesthetics to the Windows desktop environment.

#### ⚙️ Infrastructure, Environment & DevOps
*   **Arch Linux on WSL:** Fine-tuning my environment via automated deployment scripts (defaulting to XFCE for core stability), and utilizing the Chaotic-AUR.
*   **Build Optimization:** Managing Docker build configurations utilizing root privileges to reliably bypass permission gates.
*   **Asset Management:** Implementing folder-based custom protocols for loading React assets seamlessly, bypassing traditional sidecar limitations.
